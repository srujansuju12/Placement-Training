an array  is a collection of multiple values of the same data type under single variable name
Each stored value is called an element
stored value in continuous memory

why do we use array?
easy to store multiple values
faster access using index numbers
better memory management
helps in performing using operations

how to declare array?
int[] array;
String[]name;
double[]marks;
